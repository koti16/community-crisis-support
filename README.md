# Community Crisis Support

A responsive web application that connects volunteers with individuals needing help during emergencies.

## Features

- Emergency help requests
- Volunteer dashboard
- Real-time updates with Firebase
- Google Maps integration
- Authentication
- Responsive UI

## Tech Stack

- React
- Vite
- Firebase
- Tailwind CSS
- Google Maps API
- React Router
- Axios
